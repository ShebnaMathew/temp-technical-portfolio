Plagiarism Detection Tool

Northeastern University, Boston, MA

Built an application to help the Course Instructors to detect similarity/plagiarism between code submitted by two students. The Application is completely written in Typescript and uses the React library to build the UI components of the project. It detects similarity between Javascript code and can be extended further to include more languages. The algorithm is based upon comparing the 4-string representation of the source codes and can be filtered as per the need (ignore comments/ check variable, function name changes/ etc.). The project followed a Waterfall Software Development Cycle and used Github as the Version control software.
